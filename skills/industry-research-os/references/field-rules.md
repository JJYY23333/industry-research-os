## Field Meanings

| Field | Meaning |
| --- | --- |
| 类型 | Document type: player, business model, supply chain, opportunity, trend, risk |
| 行业 | Current industry |
| 地区 | Research region |
| 时间范围 | Main time range covered by the document |
| 最后更新 | Last update date |
| 状态 | 草稿, 待验证, 已验证, 过期需更新 |
| 置信度 | 高, 中, 低 |
| 一句话结论 | The most important conclusion |
| 核心事实 | Source-supported facts only |
| 分析判断 | Reasoned interpretation from facts |
| 关键指标 | Metrics such as GMV, revenue, margins, order density, retention |
| 相关玩家 | Related companies, platforms, brands |
| 风险与不确定性 | Data gaps, policy risk, competitive changes |
| 待验证问题 | Questions for follow-up verification |
| 来源 | Links, filings, announcements, reports, articles |

## Claim Labels

- 事实: directly supported by a source.
- 判断: reasoned interpretation from facts.
- 假设: plausible but not verified.
- 待验证: must be checked later.

## Source Grades

| Grade | Meaning |
| --- | --- |
| A | Financial reports, SEC/HKEX filings, company announcements, regulatory documents |
| B | Securities reports, official media, industry associations, research institutes |
| C | Media reports, industry articles, interviews, encyclopedic references |
| D | Personal judgment, social discussion, unverified clues |
